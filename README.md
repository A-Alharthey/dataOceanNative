A simple react native expo App for DataOcean

to run the project

1. **Clone the repository**
```bash
git clone https://github.com/A-Alharthey/dataOceanNative.git
```

2. **navigate to project repository**
```bash
cd dataOceanNative
```

3. **install dependencies**
```bash
npm install
```

4. **run the app with**
```bash
npx expo start
```
